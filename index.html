Вартовий робить дані відкритими

Відсутність системи для робзробки планів евакуації обмежує цивільне населення у доступі до інформації. Вартовий допоможе зробити дані публічними та надасть доступ через API та графічний інтерфейс до даних.

  • План заходів з організації, проведення та забезпечення евакуації.
  • План евакуації у вигляді мап.
  • Порядок оповіщення осіб у разі настання надзвичайних ситуацій.
  • Збірні пункти евакуації.
  • Особливості планування евакуації осіб з інвалідністю та інших маломобільних груп населення.
  • Перелік ресурсів задіяних у евакуації.
